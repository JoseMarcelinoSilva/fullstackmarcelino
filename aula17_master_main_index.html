aula17 - Jogando nova funcionalidade no master/main

O que é o git merge?


O git merge pega as alterações de uma branch e as incorpora em outra
branch. É a forma mais comum de integrar código no Git.



Sintaxe básica



git merge <branch-que-você-quer-trazer>

Importante: Você precisa estar na branch que vai receber as alterações.



Exemplo prático: jogar funcionalidade na main



# 1. Vá para a branch principal (main/master)
git checkout main

# 2. Certifique-se de que está atualizada
git pull origin main

# 3. Faça o merge da branch da funcionalidade novamente
git merge feature/nova-funcionalidade
